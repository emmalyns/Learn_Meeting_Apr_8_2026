## Steps
git 

git init (U next to file name means untracked)

git add Readme.md (adds the Readme.md file, now there is A instead of U, A means its being added)

git status (tells what is being committed, if we wanted to remove something we would do git rm fileName)

git commit

(now write what the changes were, First commit.)

OR do git commit -m "First commit."
(now the A disappears)

say we removed something but didnt mean to:

git rm GitTutorial.md

git restore GitTutorial.md

## How to see differences over time
click on the third icon on the left in vscode. 
it looks like 3-4 circles with lines. 

then near the bottom you should see the different commits.
